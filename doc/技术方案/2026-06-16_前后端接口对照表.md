## 如何更新本文档

每新增一个后端 Controller 方法或前端 API 函数，运行以下 Python 脚本校验一致性（差异会 fail，需要人工同步）：

```bash
cd /Users/jialeiwang/claudeProject/AgentOps
python3 - <<'PY'
import re, glob
controllers = glob.glob("server/adapter/src/main/java/com/agent/ops/adapter/**/*Controller.java", recursive=True)
backend = set()
for c in controllers:
    with open(c) as f: text = f.read()
    base_m = re.search(r'@RequestMapping\("(/api/[^"]+)"\)', text)
    if not base_m: continue
    base = base_m.group(1)
    for m in re.finditer(r'@(Get|Post|Put|Delete|Patch)Mapping\("([^"]+)"\)', text):
        backend.add(base + m.group(2))

files = glob.glob("frontend/src/api/*.ts")
frontend = set()
for f in files:
    with open(f) as fp: text = fp.read()
    for m in re.finditer(r"url:\s*'([^']+)'", text):
        url = m.group(1)
        if not url.startswith('http'):
            # 加上 /api 前缀对齐
            frontend.add(f"/api{url}" if not url.startswith('/api/') else url)

missing = backend - frontend
extra = frontend - backend
print(f"后端: {len(backend)}  前端: {len(frontend)}")
print(f"缺失: {len(missing)}")
for m in sorted(missing): print("  -", m)
print(f"多出: {len(extra)}")
for m in sorted(extra): print("  +", m)
PY
```

**CI 建议**：可把该脚本接入 PR 检查，差异非空则 fail，避免后续接口遗漏。
